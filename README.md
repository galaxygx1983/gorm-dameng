# GORM DaMeng Skill

达梦数据库(DM8) GORM 驱动开发指南 - Claude Code Skill

## 概述

`gorm-dameng` 是达梦数据库(DM8)的 GORM v2 方言包，基于达梦官方 Go 驱动二次开发，提供开箱即用的 ORM 支持。

## 安装

```bash
go get -d github.com/godoes/gorm-dameng
```

## 功能特性

- ✅ 标准 Go Module 安装
- ✅ 完整的数据库迁移功能
- ✅ 达梦特有语法支持（IDENTITY、MERGE INTO）
- ✅ 字符长度配置选项
- ✅ 连接字符串工具函数
- ✅ MIT 许可证（非GPL）
